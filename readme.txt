*********************** Instructions to run webapp ***********************

1. Please install python3 on your system

2. Add python to environment variables

3. Create an evironment using the following command -> virtualenv THAT_env

4 Activate the evironment using the following command  
   -> .\THAT_env\Scripts\activate   (For Windows)
   -> source THAT_env\bin\activate  (For Ubuntu)

5. Run command -> pip3 install -r requirements.txt

6. Run command -> python3 application.py

7. Click on localhost link generated after execution of previous command


********************************* End  ************************************
